Overview

Base class for all Kawaii applications. Inherit from this class to create a modular application.

Examples:

my_app.rb

require 'kawaii'
class MyApp < Kawaii::Base
  get '/' do
    'Hello, world'
  end
end

config.ru

require 'my_app.rb'
run MyApp

Overview

MVP controller. Define actions and map to them using regular routing functions.

Examples:

Routing to controllers

class HelloWorld < Kawaii::Controller
  def index
    'Hello, world'
  end

  def show
    @id = params[:id]
    render('show.html.erb')
  end
end

get '/', 'hello_world#index'
